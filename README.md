📄 DOCX 无损分割工具 v6.2

Docx Splitter — 完整修复版

一个用于 按页数、按段落、按标题智能分割 DOCX 文件 的 Python 工具。
支持 保留排版、图片、表格、页眉页脚，尤其适合 教材、大文档、学习资料拆分。

✨ 功能特点
✔ 1. 按页数分割（完美保留样式）

调用 Word COM接口（pywin32）

完整保留：

图片

表格

样式

页眉页脚

极速稳定

✔ 2. 按段落分割（python-docx）

深度复制段落 XML 节点

不破坏原格式

支持跳过空段落

可按自定义段数量切分

✔ 3. 智能标题分割

自动识别标题：

Heading 1/2/3

“第一章”“一、”“1、”“(1)” 等结构化标题
可选择按几级标题进行智能切分。

✔ 4. 文档自动分析

可自动检测：

页数（需 Word）

总段落

有效段落

自动识别标题数量

🛠️ 安装
1. Python 依赖
pip install python-docx pywin32

2. 环境要求
功能	需要 Word?	系统
按页分割	✔ 必须	仅 Windows
按段落分割	❌ 不需要	全平台
智能标题分割	❌ 不需要	全平台
🚀 使用说明

运行程序：

python split_docx.py


启动后可选择模式：

[1] 📄 按页数分割
[2] 📝 按段落分割
[3] 🧠 按标题智能分割

输入文件路径: xxx.docx

2. 按段落分割示例

3. 按标题分割示例

⚠ 注意事项
1. 如果你要 完全保留格式 ⇒ 请使用 按页分割

即使标题/段落分割也会保留绝大多数样式，但页眉页脚可能无法完全复制（python-docx 限制）。

2. 按页分割必须满足：

Windows

已安装 Microsoft Word

已安装 pywin32

3. 超大文档（3000+ 段）建议：

先使用智能标题分割

❓ FAQ
Q1: 分割后的文档是否完全无损？

按页：100% 无损

按段落与标题：约 95% 无损（页眉页脚无法复制）

Q2: 支持 PDF 吗？

不支持，请先使用 Acrobat 或其他工具转换为 DOCX。

Q3: Word 会闪一下吗？

不会，运行在后台（Visible=False）。
